# Design and Development of a Planetary Gearbox Assembly

**Year:** 2025  
**Software:** CATIA V5, DMU Kinematics

## Overview

This project focuses on the design, assembly, and kinematic validation of a planetary gearbox used in power transmission systems. The gearbox was developed in CATIA V5 with emphasis on accurate component modeling, assembly integration, motion verification, and manufacturing documentation.

---

## Objectives

- Design a complete planetary gearbox assembly.
- Perform gear ratio calculations and torque analysis.
- Model all gearbox components using CATIA V5.
- Validate gear motion using DMU Kinematics.
- Generate engineering documentation for manufacturing.

---

## Key Features

- Performed gear ratio calculations, torque analysis, and component sizing to meet functional and performance requirements.
- Developed detailed 3D parametric models of the sun gear, planet gears, ring gear, planet carrier, shafts, bearings, and housing.
- Created a fully constrained gearbox assembly using CATIA V5 Assembly Design.
- Validated gear motion and rotational relationships using DMU Kinematics.
- Generated exploded assembly views, engineering drawings, and Bill of Materials (BOM).
- Designed the assembly with emphasis on manufacturability and assembly integration.

## Design Workflow

1. Performed gearbox design calculations.
2. Designed individual components in CATIA V5 Part Design.
3. Created the complete assembly using Assembly Design.
4. Applied assembly constraints.
5. Verified component alignment and clearances.
6. Simulated gearbox motion using DMU Kinematics.
7. Generated engineering drawings and BOM.
