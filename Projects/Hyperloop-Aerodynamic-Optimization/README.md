# Hyperloop Aerodynamic Optimization

## Overview
This project investigates the influence of Hyperloop pod geometry on aerodynamic drag and confined flow characteristics using Computational Fluid Dynamics (CFD). Different pod shapes were analyzed to evaluate their aerodynamic performance and identify a design with reduced drag.

## Objectives
- Analyze the aerodynamic performance of different Hyperloop pod geometries.
- Compare drag coefficients for various pod shapes.
- Study pressure and velocity distributions around the pod.
- Evaluate the effects of blockage ratio and confined flow.

## Software Used
- SolidWorks
- ANSYS Fluent

## Methodology
1. Created multiple Hyperloop pod models in SolidWorks.
2. Generated the computational domain and mesh.
3. Applied boundary conditions in ANSYS Fluent.
4. Performed CFD simulations.
5. Compared aerodynamic results for each pod configuration.

## Results
- Compared drag values for different pod geometries.
- Evaluated pressure and velocity contours.
- Identified the most aerodynamically efficient pod design.
